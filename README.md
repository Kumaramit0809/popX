# PopX App

A responsive React.js application built by recreating the provided Adobe XD design.

## Features

- User Registration Page
- Login Page
- Account Settings Page
- Responsive UI Design
- React Router Navigation
- Modern React Functional Components

## Tech Stack

- React.js
- React Router
- JavaScript (ES6+)
- CSS3
- Vite

## Installation

Clone the repository:

```bash
git clone https://github.com/Kumaramit0809/popX.git
```

Navigate to the project folder:

```bash
cd popX
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
